# Tidle  
*A simple, lightweight task management app that is designed to fit in your workflow.*

## Key Features  

### 1. Drag‑and‑Drop 
Organize to‑dos across sections (Todo, In Progress, Completed) with simple drag‑and‑drop control.  
<img width="200" alt="image" src="https://github.com/user-attachments/assets/5a6988fb-9a33-48cd-9c4b-e8211641c836" />

### 2. Reminders
Set timely alerts so important tasks never slip through.
<img width="200" alt="image" src="https://github.com/user-attachments/assets/1e806d14-a015-4518-a786-6f7b8ce1b32e" />

### 3. Subtasks, Notes, Attachments
Add subtasks, notes, and attachments to keep every task organized, detailed, and easy to follow.
<img width="200" alt="image" src="https://github.com/user-attachments/assets/dccdbceb-b74d-461e-ad55-c676d710a198" />

### 4. Calendar Preview  
Switch between week and month views and pick dates to visualize upcoming commitments.  

### 5. Analytics Dashboard
See productivity at a glance—weekly completion scores, daily progress bars, and category breakdowns help you spot trends and stay accountable.

Separate work, personal, and study workspaces to organize your work accordingly.<br/>
