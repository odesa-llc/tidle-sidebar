# Tidle for Obsidian

Embed the Tidle task manager inside Obsidian so you can manage tasks alongside your notes.

This plugin opens the Tidle web app in an Obsidian pane using an embedded iframe.

## Features (Obsidian)

- Open Tidle inside Obsidian in a dedicated view
- Ribbon icon and command palette command
- Optional auto-open on launch (if enabled in the plugin)
- Desktop only

## How it works

This plugin loads the Tidle web application from:

- https://tidle.app/main

The plugin itself does not read, modify, or transmit any data from your Obsidian vault.  
All task data, authentication, and storage are handled by the Tidle web application.

## Installation

### From Obsidian Community Plugins
1. Open Obsidian
2. Settings → Community plugins
3. Search for **Tidle**
4. Install and enable

## Usage

- Click the Tidle ribbon icon
- Or open the command palette and run **Open Tidle**

Tidle will open in a pane. You can keep it docked as a sidebar tab if you like.

## Privacy and data usage

This plugin loads external content from `https://tidle.app/main`.

- The plugin does not access, read, or index your vault files.
- The plugin does not send vault content anywhere.
- Network activity is limited to loading the embedded Tidle web application.

Any task data, authentication, cookies, analytics, and storage are handled by the Tidle web application and governed by Tidle’s privacy policy:
- https://tidle.app/privacy-policy

By using this plugin, you are accessing the Tidle web application within Obsidian.

## Security notes

- No Obsidian vault files are accessed by this plugin
- No local files are transmitted outside of Obsidian
- The plugin UI is an embedded web view (iframe)

## Compatibility

- Desktop only
- Requires Obsidian version 0.15.0+
