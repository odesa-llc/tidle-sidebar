# Tidle  
*A simple, lightweight task management app that is designed to fit in your workflow.*

## Key Features  

### 1. Drag‑and‑Drop 
Organize to‑dos across sections (Todo, In Progress, Completed) with simple drag‑and‑drop control.  
<img width="200" alt="image" src="https://github.com/user-attachments/assets/59d90fa3-00c0-47a5-8bb0-2c418e24a35b" />

### 2. Files, Links, and Tagging 
Attach files or links and automatically tag tasks based on keywords—Tidle keeps repeating work and categorization handled for you.  
<img width="200" alt="image" src="https://github.com/user-attachments/assets/41a74e41-3abc-4d89-87a2-c0de87270c39" />

### 3. Calendar Preview  
Switch between week and month views and pick dates to visualize upcoming commitments.  
<img width="200" alt="image" src="https://github.com/user-attachments/assets/e1c7391f-954d-4518-96c9-5f42ccde1dd8" />

### 4. Analytics Dashboard
See productivity at a glance—weekly completion scores, daily progress bars, and category breakdowns help you spot trends and stay accountable.  
<img width="200" alt="image" src="https://github.com/user-attachments/assets/2df74eda-77da-4b14-97a4-0d75a30baba3" />

### 5. Workspace & Settings Panel
Separate work, personal, and study workspaces to organize your work accordingly.<br/>
